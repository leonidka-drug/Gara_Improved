Это улучшенная версия вкладки p2p с сайта garantex.io, которая:


1. Обновляет данные рынка автоматически каждые 4 секунды
2. Даёт возможность подсветить ваше и чужие объявление для удобства отслеживания позиции на рынке.


> Чтобы начать пользоваться этим инструментом, вам нужно скачать архив [этого репозитория](https://github.com/leonidka-drug/Gara_Improved) , распаковать его, и открыть файл index.html
